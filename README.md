# Netbox to NAK

## Description

Scripts for generating YAML configuration files for Network Administration Kit (https://github.com/tmshlvck/nak) from Netbox (https://github.com/netbox-community/netbox).

## Requirements

  - Python 3.5+
  - PyYAML
  - requests
